# React Start

一个简易的后台系统前端脚手架

### 环境依赖
- 安装[nodejs](https://nodejs.org/), 推荐使用[nvm]()安装node


### 获取项目

```
$ git clone xxx
```
### 安装代码依赖包
```
$ npm install
$ npm install roadhog -g
```

### 本地启动
```
$ npm start   # 访问 http://localhost:8000
```

### 打包
#### alpha环境
```
$ npm run build:alpha # 输出文件在dist/目录下
```

#### 生产环境
```
$ npm run build  # 输出文件在dist/目录下
```
